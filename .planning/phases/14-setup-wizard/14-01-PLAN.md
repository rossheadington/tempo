    `_FILE_MODE = 0o600` module-level constant (same as tokens.py).

    `_QUOTE_REQUIRED_CHARS = (" ", "$", "\t", "#")` module-level constant.

    `_quote_value(value: str) -> str` (module-private helper):
    - If the value contains any of `_QUOTE_REQUIRED_CHARS`, return `f'"{value}"'` (wrap in double quotes; do NOT escape `$` — dotenv-compatible simplest rule).
    - If the value is empty, return `""` (preserves the dotenv convention of `KEY=` for empty).
    - Otherwise return the value unchanged.

    `read_env(path: Path) -> dict[str, str]`:
    - If `not path.exists()` → return `{}`. NEVER raises FileNotFoundError.
    - Read the file as UTF-8. Iterate lines. For each non-comment (does not start with `#` after lstrip), non-blank line:
      - Find the first `=`. If none, skip the line silently.
      - Key = everything before the `=`, stripped. If key is empty, skip.
      - Value = everything after the first `=`. Strip trailing newline. Strip surrounding double quotes IF the value starts AND ends with `"` AND length ≥ 2.
      - Last value wins on duplicate keys (later writes overwrite earlier reads).
    - Returns the dict.

    `atomic_write_env(path: Path, updates: dict[str, str], delete_keys: frozenset[str] | set[str] = frozenset()) -> None`:
    - Mirror `TokenStore.save` exactly. Steps:
      1. Read existing file (if any) into a `list[str]` of lines (preserve `\n` so a re-write is identical to a no-op write).
      2. Walk the lines. For each line:
         - If the line is blank or comment-only (`lstrip().startswith("#")`), keep verbatim.
         - Else parse `key=value` (first `=`); if the key matches a key in `delete_keys`, drop the line. If the key matches a key in `updates`, rewrite the line as `f"{key}={_quote_value(updates[key])}\n"` and mark the key as "consumed" (so we don't re-append it later).
         - Else preserve verbatim.
      3. For each key in `updates` NOT consumed in step 2:
         - If the last preserved line does not end with a blank-line separator, append `"\n"` first.
         - Append `f"{key}={_quote_value(value)}\n"`.
      4. Compute the destination directory (`path.parent`); `mkdir(parents=True, exist_ok=True)`.
      5. `mkstemp(dir=path.parent, prefix=f".{path.name}.", suffix=".tmp")` → get `(fd, tmp_name)`.
      6. `os.fchmod(fd, 0o600)`.
      7. With `os.fdopen(fd, "w", encoding="utf-8")` write the lines, `flush()`, `os.fsync(handle.fileno())`.
      8. `os.replace(tmp_path, path)`.
      9. Best-effort `os.fsync` on `path.parent` (open dir read-only, fsync, close). Swallow OSError.
      10. `os.chmod(path, 0o600)` (belt-and-braces).
      11. On ANY exception in steps 5-10, `tmp_path.unlink(missing_ok=True)` and re-raise.
    - The function NEVER logs the value of any update. Stay quiet.

    Implementation note: do NOT use python-dotenv. Stdlib only (`os`, `tempfile`, `pathlib`). The parser is intentionally simple (last value wins, no variable interpolation, no `export ` prefix support); the .env files we own are entirely human-edited and follow the simple `KEY=value` shape documented in `.env.example`.
